# BO26EB-40
Bachelor oppgave: Tilstandsovervåking av bolter basert på akselerometerdata med maskinlæring.

Dette prosjektet inneholder all kode for bachelor oppgaven.

Prosjektet er delt opp i 4 deler:

Server - leser data fra akselerometeret og sender det på valgt IP adresse.

Klient - leser data fra valgt IP adresse og lagrer på valgt plass.

1D CNN modell - kode for trening av modell og ferdig modell med justerte hyperparametere.

Random Forest modell - kode for trening av modell og ferdig modell.
